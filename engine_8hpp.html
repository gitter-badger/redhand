◆ REDHAND_VERSION_NUMBER

#define REDHAND_VERSION_NUMBER   0

This integere defines the version number.

You should use it to check if the version of your game is compatible with the engine. The first public version is 1 and every release will be 1 higher. Look in the documentation to see which features are present in which version of the engine.
